Finexa Capital Financial Data Pipeline (Capstone Project)

Developed an automated ETL pipeline for a fictional investment firm using Apache Airflow, Azure Databricks, and PostgreSQL. The pipeline collects financial data via API, cleans and transforms it, and loads it securely into a database. Key features include incremental data loading, secure authentication (API keys, tokens, Azure Entra ID), and robust logging/error tracking to simulate real-world financial data workflows.
